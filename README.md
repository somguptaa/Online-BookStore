# Online-BookStore

This is the backend of my project. Currently, it has only **Register** and **Login** modules implemented, I'm developing other things 

## API Endpoints
- `POST /userregisters` → Register a new user
- `POST /userlogin` → Authenticate user

Future updates will include:
- Book management (CRUD for books)
- Payment pages
- Frontend development

## Tech Stack
- Java 21
- Spring Boot 3.5.6
- Oracle 21c
- Maven
- Spring Data JPA
- REST APIs
